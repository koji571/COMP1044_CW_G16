# COMP1044_CW_G16

A Guide as to how to run the programme

Requirements:
1. XAMPP
downloadable at (https://www.apachefriends.org/download.html)
2. Web Browser


Instructions:
1. Ensure that the latest version of XAMPP is downloaded
2. Launch XAMPP and ensure "MYSql Database" and "Apache Web Server" is running
3. Download all the source code from the repository
4. Transfer all the files from the repository into your XAMPP working folder, default folder “E:\xampp\htdocs” for Windows users.
5. Import the LibraryDB.sql into your local MYSql server.
6. To begin using the library web server, type "http://localhost/Login.php" in your search bar
7. Done
